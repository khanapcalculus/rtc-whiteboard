# RTC Whiteboard

A real-time collaborative whiteboard application built with React and TypeScript.

## Features

- Drawing tools (pen, shapes, text)
- Color and stroke width selection
- Real-time collaboration
- Undo/Redo functionality
- Image upload support
- Pan and zoom

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The application is automatically deployed to GitHub Pages when changes are pushed to the main branch. You can also manually deploy using:

```bash
npm run deploy
```
