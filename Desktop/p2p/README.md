# P2P Whiteboard

A WebRTC-based peer-to-peer whiteboard application for online teaching. This application allows teachers and students to connect directly for one-to-one sessions with video, audio, and a collaborative whiteboard.

## Features

- Real-time peer-to-peer communication using WebRTC
- Interactive whiteboard with drawing tools
- Video and audio communication
- Screen sharing capability
- Low-latency direct connections between peers
- Simple room-based connection system

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **WebRTC**: PeerJS library
- **Canvas**: Fabric.js for the whiteboard functionality
- **Signaling Server**: Node.js with Socket.IO

## Project Structure

- `client/`: Frontend code (deployed to GitHub Pages)
- `server/`: Signaling server code (deployed to Render.com)
